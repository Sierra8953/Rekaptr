## 2024-05-24 - Tooltip Redundancy
**Learning:** Adding tooltips to icon-only buttons is crucial for accessibility, but when an icon is already paired with explicit, immediately adjacent visible text (e.g. an "Add Source" button that has both the plus icon and the text "Add Source" beneath it), adding a tooltip is redundant and clutters the UI.
**Action:** Before adding a tooltip to an icon, verify if there is accompanying label text in the UI layout that serves the same descriptive purpose.
