# Rekaptr Roadmap

Detailed roadmaps for each version are in the `roadmap/` directory:

- [v0.2.0 — Stability](roadmap/v0.2.0-stability.md) — Fix crashes, data loss, and silent failures
- [v0.3.0 — Reliability](roadmap/v0.3.0-reliability.md) — DB migrations, deterministic file ops, pipeline validation
- [v0.4.0 — Performance](roadmap/v0.4.0-performance.md) — Config caching, async UI, task tracking
- [v0.5.0 — Polish](roadmap/v0.5.0-polish.md) — Error reporting, UX improvements, keyboard shortcuts
- [v0.6.0 — Audio](roadmap/v0.6.0-audio.md) — Waveforms, level meters, noise gate, mixer
- [v0.7.0 — Recording Features](roadmap/v0.7.0-recording-features.md) — Instant replay, pause/resume, overlays
- [v0.8.0 — Export & Sharing](roadmap/v0.8.0-export-sharing.md) — Smart cut, presets, batch export, uploads
- [v0.9.0 — Ecosystem](roadmap/v0.9.0-ecosystem.md) — Auto-updater, installer, crash reporting, plugins
- [v1.0.0 — Release](roadmap/v1.0.0-release.md) — Test suite, accessibility, localization, distribution
